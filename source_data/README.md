# Source data

This directory contains processed source tables supporting the manuscript
figures, supplementary figures and summary claims.

## Main analysis blocks

- `summary_tables/`: manuscript-level summary tables, including the five-seed
  8UN4 closed-loop summary.
- `fig2_controlled_validation/`: posterior-controlled acquisition and timing
  comparisons.
- `fig3_dockstring_screening/`: matched Dockstring finite-pool screening
  trajectories and endpoint summaries.
- `fig4_domain_prior_and_generators/`: natural-product chemical-space and
  generator/candidate-pool analyses.
- `fig6_8un4_closed_loop/`: processed 8UN4 docking-prioritization source
  tables.
- `supplementary/`: source tables for supplementary computational diagnostics
  and cell-line profiling.
