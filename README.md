# Exact-descriptor active screening

This repository contains code and processed source data supporting the
manuscript "Exact-descriptor active screening for finite-pool molecular
prioritization".

The public release contains processed source tables, acquisition code,
surrogate benchmarks, docking-prioritization workflows and cell-line profiling
source tables.

## Repository layout

- `method_code/`: acquisition, surrogate, generator, docking-oracle and
  benchmarking code used for the manuscript analyses.
- `source_data/`: processed source tables for the main figures, supplementary
  figures and summary tables.

The source tables are organized by figure and analysis block so that readers
can inspect the numerical inputs behind the reported computational results.
The method code captures the finite-pool active-screening workflow used in the
manuscript.
